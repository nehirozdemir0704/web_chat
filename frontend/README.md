# Frontend

This folder contains the static files for the Discord-like chat SPA.

- `public/index.html`: Main HTML entry point, loads Tailwind CSS and app.js.
- `public/app.js`: All SPA logic, UI, and WebSocket communication.

**How to run:**
- The backend (Node.js server) serves these files automatically.
- Just run `node backend/server.js` from the project root and open http://localhost:3000 in your browser.
